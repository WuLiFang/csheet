归档打包
=======================

支持将色板打包为离线网页应用供无法访问色板服务器的用户查看。

离线应用为当前收藏数据的快照，仅包含转码后的图像供查看，不包含原始文件。

所有涉及写入服务器数据库的操作均在离线版中无法使用，并且不支持实时更新。

访问服务器地址 ``/archive`` 端点即可下载压缩包。

路径参数
--------------

通过网址查询参数指定要下载的收藏，支持以下参数：

tag_or?: string[]

  匹配任意标签

tag_and?: string[]

  匹配所有标签

origin_prefix?: string

  匹配来源前缀

presentation_count_gt?: number

  匹配包含的展示数量大于此值的收藏

title?: string

  生成的色板标题，用于文件名和文档标题。不填则通过当前时间生成。

file_path_format?: string

  文件路径格式。 windows 或 unix，默认和服务器使用相同格式。
