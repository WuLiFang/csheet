文件夹
=========================

从文件夹进行收集。

.. uml::

  start 
  if (路径不匹配包含路径)
    end
  elseif (路径匹配排除路径)
    end
  endif
  if (路径是文件夹)
    :处理文件夹下的内容;
  elseif (是支持的文件类型)
    :创建或更新对应收集;
  endif
  stop


来源格式: ``folder:{文件绝对路径}``，
Windows 路径将全部转化为小写，因为 Windows 文件系统不区分大小写。



路径匹配
-------------------

可设置包含和排除路径，不包含或者被排除的文件夹下的文件将被忽略。
