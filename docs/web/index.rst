==================================

.. image:: _images/chrome_2020-09-03_11-57-08.png

可访问服务器部署的地址来浏览色板。

本软件是一个单网页应用。

如果不带参数进行访问，会自动恢复上次使用的状态。

收藏和展示均为实时数据更新，无需手动刷新页面。
