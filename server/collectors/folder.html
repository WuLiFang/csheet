文件夹¶

从文件夹进行收集。

来源格式: folder:{文件绝对路径}， Windows 路径将全部转化为小写，因为 Windows 文件系统不区分大小写。

路径匹配¶

可设置包含和排除路径，不包含或者被排除的文件夹下的文件将被忽略。
