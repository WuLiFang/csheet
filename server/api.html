GraphQL API¶

系统提供 GraphQL API。

HTTP 路由 路径为 /api

实时订阅¶

支持通过 Websocket 进行 GraphQL 订阅，订阅路径也是 /api。

编辑器¶

直接用浏览器访问（HTTP GET Accept 头匹配 text/html）时， 将显示 Graphql Playground 编辑器。

持久查询¶

支持 APQ 来提升查询性能。

发送请求时可用查询摘要代替查询文本。
