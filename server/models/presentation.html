展示¶

从收集的文件中获取到的展示方式。

每个文件可对应任意个数的展示方式。

结构¶

本节的相对路径都相对于 文件存储 files 目录。

Type Type

展示类型，和 转码 类型对应。

Raw string

原始文件绝对路径。

Thumb string

缩略尺寸文件相对路径。

ThumbSuccessTag string

缩略文件 转码 成功的原始文件 内容标签。

ThumbErrorTag string

缩略文件 转码 失败的原始文件 内容标签。

Regular string

常规尺寸文件相对路径。

RegularSuccessTag string

常规文件 转码 成功的原始文件 内容标签。

RegularErrorTag string

常规文件 转码 失败的原始文件 内容标签。
